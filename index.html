Configurar banco de dados em tempo real

Para compartilhar com qualquer pessoa via link (como Google Sheets), cole a URL do seu Firebase Realtime Database. É gratuito e leva 2 minutos.

Como criar em 2 minutos (gratuito):
1. Acesse console.firebase.google.com → Criar projeto
2. Menu lateral: Build → Realtime Database → Criar banco
3. Escolha "Iniciar no modo de teste"
4. Copie a URL gerada (ex: https://meu-projeto-default-rtdb.firebaseio.com)
5. Cole acima e clique em Conectar — todos que abrirem o arquivo verão os mesmos dados!
